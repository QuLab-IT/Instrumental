APPLy:DC [<frequency|DEFault> [,<amplitude>|DEFault> [,<offset>] ]]

Output a dc voltage with the level specified by the offset parameter. You can set the dc voltage to any value between ±5 Vdc into 50 ohms or ±10 Vdc into an open circuit. The dc voltage is output as soon as the command is executed.
· The frequency and amplitude parameters have no effect for this command but you must specify a value or "DEFault". If you specify a frequency and amplitude, they have no effect on the dc output, but the values are remembered when you change to a different function. The following statement shows the use of the APPLy command for a dc output.

  APPL:DC DEF, DEF, -2.5
