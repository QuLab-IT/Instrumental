APPLy:NOISe [<frequency|DEFault> [,<amplitude> [,<offset>] ]]

Output Gaussian noise with the specified amplitude and dc offset. The waveform is output as soon as the command is executed.
· The frequency parameter has no effect for this command but you must specify a value or "DEFault" (the noise function has a 50 MHz bandwidth). If you specify a frequency, it has no effect on the noise output, but the value is remembered when you change to a different function. The following statement shows the use of the APPLy command for noise.

  APPL:NOIS DEF, 5.0, 2.0
