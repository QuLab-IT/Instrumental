Baud Rate Selection (RS-232)

You can select one of several baud rates for RS-232 operation. When shipped from the factory, the baud rate is set to 57,600 baud.
You can set the baud rate from the front panel only.
· Select one of the following: 300, 600, 1200, 2400, 4800, 9600, 19200, 38400, 57600 (factory setting), or 115200 baud.
· The baud rate selection is stored in non-volatile memory and does not change when power has been off or after a remote interface reset.
· Front-Panel Operation: Pressand then select the Baud Rate softkey from the "I/O" menu.
