Burst Mode Commands

Use the APPLy command or the equivalent FUNC, FREQ, VOLT, and VOLT:OFFS commands to configure the waveform. For internally-triggered bursts, the minimum frequency is 2 mHz. For sine and square waveforms, frequencies above 25 MHz are allowed only with an "infinite" burst count.
