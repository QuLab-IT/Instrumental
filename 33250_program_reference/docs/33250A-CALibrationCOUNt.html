CALibration:COUNt?

Query the instrument to determine the number of times it has been calibrated. Note that your instrument was calibrated before it left the factory. When you receive your instrument from the factory, be sure to read the count to determine its initial value.
· The calibration count is stored in non-volatile memory. The count increments up to a maximum of "65,535" after which it rolls over
to "0". Since the value increments by one for each calibration point, a complete calibration may increase the value by many counts.
