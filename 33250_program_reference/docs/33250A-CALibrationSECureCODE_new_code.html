CALibration:SECure:CODE <new code>

Enter a new security code. To change the security code, you must first unsecure the function generator using the old security code, and then enter a new code. The security code is stored in non-volatile memory.
· The calibration code may contain up to 12 characters. The first character must be a letter (A-Z), but the remaining characters can be letters, numbers (0-9), or the underscore character (" _ "). You do not have to use all 12 characters, but the first character must always be a letter.
