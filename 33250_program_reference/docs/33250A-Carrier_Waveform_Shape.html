Carrier Waveform Shape

· AM carrier shape: Sine, Square, Ramp, or Arbitrary waveform. The default is Sine. You cannot use pulse, noise, or dc as the carrier waveform.
· Front-Panel Operation: Press any of the front-panel function keys exceptor. For arbitrary waveforms, pressand then choose the Select Wform softkey to select the active waveform.
· Remote Interface Operation:

 FUNCtion:SHAPe {SINusoid|SQUare|RAMP|USER}

You can also use the APPLy command to select the function, frequency, amplitude, and offset with a single command.
