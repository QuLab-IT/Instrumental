DISPlay {OFF|ON}
DISPlay?

Disable or enable the function generator front-panel display. When it is disabled, the front-panel display is blanked (however, the bulb used to backlight the display remains enabled). The DISP? query returns "0" (OFF) or "1" (ON).
· With the front-panel display disabled, there will be some improvement in command execution speed from the remote interface.
· Sending a message to the front-panel display from the remote interface (DISP:TEXT command) overrides the display state. This means that you can display a message even if the display is disabled (remote interface errors are always displayed even if the display is disabled).
· The display is automatically enabled when power is cycled, after an instrument reset (*RST command), or when you return to local (front panel) operation. Press thekey or execute the IEEE-488 GTL (Go To Local) command from the remote interface to return to the local state.
· The display state is saved when you store the instrument state using *SAV command. When you recall the instrument state using *RCL command, the front-panel display will return to the previous state.
