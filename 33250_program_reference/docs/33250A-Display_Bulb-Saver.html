Display Bulb-Saver

The front-panel display bulb will normally turn off and the screen will go blank after one hour of inactivity. For certain applications, you may want to disable the bulb-saver feature. This feature is available from the front panel only.
· The bulb-saver setting is stored in non-volatile memory and does not change when power has been off or after a remote interface reset. When shipped from the factory, the bulb-saver mode is enabled.
· Front-Panel Operation: Pressand then select the Scrn Svr (screen saver) softkey from the "System" menu.
