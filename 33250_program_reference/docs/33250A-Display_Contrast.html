Display Contrast

To optimize the readability of the front-panel display, you can adjust the contrast setting. This feature is available from the front panel only.
· Display contrast: 0 to 100. The default is 50.
· The contrast setting is stored in non-volatile memory and does not change when power has been off or after a remote interface reset.
· Front-Panel Operation: Pressand then select the Display Contr softkey from the "System" menu.
