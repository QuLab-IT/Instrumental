FM:INTernal:FREQuency {<frequency>|MINimum|MAXimum}
FM:INTernal:FREQuency? [MINimum|MAXimum]

Set the frequency of the modulating waveform. Used only when the Internal modulation source is selected (FM:SOUR INT command). Select from 2 mHz to 20 kHz. The default is 10 Hz. MIN = 2 mHz.
MAX = 20 kHz. The :FREQ? query returns the internal modulating frequency in hertz.
