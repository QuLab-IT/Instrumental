FSKey:FREQuency {<frequency>|MINimum|MAXimum}
FSKey:FREQuency? [MINimum|MAXimum]

Set the FSK alternate (or "hop") frequency. Select from 1 mHz to 80 MHz (limited to 1 MHz for ramps and 25 MHz for arbitrary waveforms). The default is 100 Hz. MIN = 1 mHz. MAX = 80 MHz. The :FREQ? query returns the "hop" frequency in hertz.
· The modulating waveform is a square wave with a 50% duty cycle.
