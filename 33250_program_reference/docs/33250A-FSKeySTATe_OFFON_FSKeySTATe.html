FSKey:STATe {OFF|ON}
FSKey:STATe?

Disable or enable FSK modulation. To avoid multiple waveform changes, you can enable FSK after you have set up the other modulation parameters. The default is OFF. The :STAT? query returns "0" (OFF) or "1" (ON).
· The function generator will allow only one modulation mode to be enabled at a time. For example, you cannot enable FSK and AM at the same time. When you enable FSK, the previous modulation mode is turned off.
· The function generator will not allow FSK to be enabled at the same time that sweep or burst is enabled. When you enable FSK, the sweep or burst mode is turned off.
