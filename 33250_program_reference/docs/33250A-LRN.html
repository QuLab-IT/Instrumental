*LRN?

Query the function generator and return a string of SCPI commands containing the current settings (learn string). You can then send the string back to the instrument to restore this state at a later time. For proper operation, do not modify the returned string before sending it to the function generator. The returned string will contain about 1,500 characters. To avoid possible instrument errors, execute the *RST command first, before sending the learn string to the instrument.
