MEMory:STATe:DELete {0|1|2|3|4}

Delete the contents of the specified storage location. If you have assigned a user-defined name to a location (MEM:STAT:NAME command), this command also removes the name that you assigned and restores the default name ("AUTO_RECALL", "STATE_1", "STATE_2", etc.). Note that you cannot recall the instrument state from a storage location that is empty. An error is generated if you attempt to recall a deleted state.
