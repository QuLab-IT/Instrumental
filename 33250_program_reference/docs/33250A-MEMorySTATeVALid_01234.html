MEMory:STATe:VALid? {0|1|2|3|4}

Query the specified storage location to determine if a valid state is currently stored in that location. You can use this command before sending the *RCL command to determine if a state has been previously stored in this location. Returns "0" if no state has been stored or if it has been deleted. Returns "1" if a valid state is stored in the specified location.
