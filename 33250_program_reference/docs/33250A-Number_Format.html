Number Format

The function generator can show numbers on the front-panel display with periods or commas for the decimal point and digits separator. This feature is available from the front panel only.

  Decimal Point: Period
  Digits Separator: Comma
      Decimal Point: Comma
      Digits Separator: None
· The number format is stored in non-volatile memory and does not change when power has been off or after a remote interface reset. When shipped from the factory, a period is used for the decimal point and commas are used for the digits separator (e.g., 1.000,000,00 kHz).
· Front-Panel Operation: Pressand then select the Number Format softkey from the "System" menu.
