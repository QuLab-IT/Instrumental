Numeric Parameters

Commands that require numeric parameters will accept all commonly used decimal representations of numbers including optional signs, decimal points, and scientific notation. Special values for numeric parameters like MINimum, MAXimum, and DEFault are also accepted. You can also send engineering unit suffixes with numeric parameters (e.g.,  Mhz or Khz). If only specific numeric values are accepted, the function generator will automatically round the input numeric parameters. The following command uses a numeric parameter:
FREQuency {<frequency>|MINimum|MAXimum}
