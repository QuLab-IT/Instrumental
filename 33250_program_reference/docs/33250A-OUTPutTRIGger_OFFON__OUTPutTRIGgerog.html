OUTPut:TRIGger {OFF|ON}
OUTPut:TRIGger?

Disable or enable the "trigger out" signal (used with burst and sweep only). When enabled, a TTL-compatible square waveform with the specified edge (OUTP:TRIG:SLOP command) is output from the rear-panel Trig Out connector at the beginning of the burst. The default is OFF. The :TRIG? query returns "0" (OFF) or "1" (ON).
