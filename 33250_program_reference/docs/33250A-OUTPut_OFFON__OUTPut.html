OUTPut {OFF|ON}
OUTPut?

Disable or enable the front-panel Output connector. The default is "OFF". When the output is enabled, thekey is illuminated on the function generator's front panel. The OUTP? query returns "0" (OFF) or "1" (ON).
· The APPLy command overrides the current OUTP command setting and automatically enables the Output connector ("ON").
· If an excessive external voltage is applied to the front-panel Output connector, an error message will be displayed and the output will be disabled. To re-enable the output, remove the overload from the Output connector and send the OUTP ON  command.
