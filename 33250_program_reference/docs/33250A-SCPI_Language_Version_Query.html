SCPI Language Version Query

The function generator complies with the rules and conventions of the present version of SCPI (Standard Commands for Programmable Instruments). You can determine the SCPI version with which the instrument is in compliance by sending a query from the remote interface.
You cannot query the SCPI version from the front panel.
· Remote Interface Operation:

 SYSTem:VERSion?

Returns a string in the form "YYYY.V", where "YYYY" represents the year of the version, and "V" represents a version number for that year (e.g., 1997.0).
