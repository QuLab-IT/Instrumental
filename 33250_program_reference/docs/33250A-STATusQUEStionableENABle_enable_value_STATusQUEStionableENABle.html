STATus:QUEStionable:ENABle <enable value>
STATus:QUEStionable:ENABle?

Enable bits in the enable register in this register group. The selected bits are then reported to the Status Byte. A *CLS (clear status) will not clear the enable register but it does clear all bits in the event register. The STATus:PRESet command clears all bits in the enable register. To enable bits in the enable register, you must write a decimal value which corresponds to the binary-weighted sum of the bits you wish to enable in the register.
The :ENAB? query returns a decimal value which corresponds to the binary-weighted sum of all bits enabled by the STAT:QUES:ENAB command.
