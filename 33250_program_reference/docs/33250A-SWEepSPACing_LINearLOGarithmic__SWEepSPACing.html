SWEep:SPACing {LINear|LOGarithmic}
SWEep:SPACing?

Select linear or logarithmic spacing for the sweep. The default is Linear. The :SPAC? query returns "LIN" or "LOG".
· For a linear sweep, the function generator varies the output frequency in a linear fashion during the sweep.
· For a logarithmic sweep, the function generator varies the output frequency in a logarithmic fashion during the sweep.
