Selecting a Trigger Source

When Sweep or Burst is enabled, the function generator will accept an immediate internal trigger, a hardware trigger from the rear-panel Trig In connector, a manual trigger from the front-panelkey, or a software (bus) trigger. By default, the internal trigger source is selected. If you want to use an external or a software trigger source, you must first select that source. For example, the following command strings sent from your computer will output a 3-cycle burst each time the rear-panel Trig In connector receives the rising edge of a TTL pulse.
BURS:NCYC 3 Set burst count to 3 cycles
TRIG:SLOP POS Set polarity to rising edge
TRIG:SOUR EXT Select external trigger source
BURS:STAT ON Enable burst mode
