Self-Test

· A power-on self-test occurs automatically when you turn on the function generator. This limited test assures you that the function generator is operational.
· A complete self-test runs a series of tests and takes approximately 15 seconds to execute. If all tests pass, you can have high confidence that the function generator is fully operational.
· If the complete self-test is successful, "Self-Test Passed" is displayed on the front panel. If the self-test fails, "Self-Test Failed" is displayed and an error number is shown. See the Agilent 33250A Service Guide for instructions on returning the instrument to Agilent for service.
· Front-Panel Operation: Pressand then select the Self Test softkey from the "Test  /  Cal" menu.
· Remote Interface Operation:

 *TST?

Returns "0" if the self-test passes or "1" if it fails. If the self-test fails, an error message is also generated with additional information on why the test failed.
