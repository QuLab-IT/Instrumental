TRIGger:DELay {<seconds>|MINimum|MAXimum}
TRIGger:DELay? [MINimum|MAXimum]

Insert a time delay between the receipt of the trigger and the start of the burst waveform (used in triggered burst mode only). Select from 0 seconds to 85 seconds. The default delay is 0. MIN = 0 seconds. MAX = 85 seconds. The :DEL? query returns the trigger delay in seconds.
