The Standard Event Register

The Standard Event register group reports the following types of events: power-on detected, command syntax errors, command execution errors, self-test or calibration errors, query errors, or the *OPC command has been executed. Any or all of these conditions can be reported to the Standard Event summary bit through the enable register. To set the enable register mask, you must write a decimal value to the register using the *ESE command.

The Standard event register is cleared when:
· You execute the *CLS command.
· You query the event register using the *ESR? command.
The Standard Event enable register is cleared when:
· You execute the *ESE 0 command.
· You turn the power on and have previously configured the function generator to clear the enable register using the *PSC 1 command. Note that the enable register will not be cleared at power-on if you have configured the function generator using the *PSC 0 command.
