Using *STB? to Read the Status Byte

The *STB? command is similar to a Serial Poll but it is processed like any other ASCII instrument command. The *STB? command returns the same result as a Serial Poll but bit 6 is not cleared as long as the enabled conditions remain.
The *STB? command is not handled automatically by the IEEE-488 bus interface hardware and will be executed only after previous commands have completed. You cannot clear an SRQ using the *STB? command.
