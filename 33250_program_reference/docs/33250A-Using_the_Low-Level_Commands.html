Using the Low-Level Commands

Although the APPLy command provides the most straightforward method to program the function generator, the low-level commands give you more flexibility to change individual parameters. For example, the following command strings sent from your computer will output a 3 Vpp sine wave at 5 kHz with a -2.5 volt offset.
FUNC SIN Select sine wave function
FREQ 5000 Set frequency to 5 kHz
VOLT 3.0    Set amplitude to 3 Vpp
VOLT:OFFS -2.5 Set offset to -2.5 Vdc
