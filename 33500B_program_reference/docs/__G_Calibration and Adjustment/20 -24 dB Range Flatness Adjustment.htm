-24 dB Range Flatness Adjustment

  1. Connect a precision AC voltmeter to measure the output amplitude as shown below. Connect the BNC cable to the Wide Band input of the Fluke 5790A.

  2. Use the precision AC voltmeter to measure the output amplitude for each setup in the table below.

    SetupNominal Signal 
     FrequencyAmplitude

    43*20 MHz0.192 VrmsFlatness for -24 dB range
    44*25 MHz0.192 VrmsFlatness for -24 dB range
    45*30 MHz0.192 VrmsFlatness for -24 dB range

    * Constants are stored after completing this setup.
  3. Using the numeric keypad or knob, adjust the displayed voltage at each setup to match the measured voltage. Select ENTER VALUE.
  4. After performing setup 45:
  1. If your calibration procedures require you to verify the adjustment just made, exit the calibration menu and perform -24 dB Range Flatness Verification.
  2. If you are making all of the adjustments and then verifying the instrument’s performance, continue with the next procedure in this section.
