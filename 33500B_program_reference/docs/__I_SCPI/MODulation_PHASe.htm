[SOURce[1|2]:]MODulation:PHASe {<angle>|MINimum|MAXimum}
[SOURce[1|2]:]MODulation:PHASe?

Sets the phase of the internal modulation source when modulating by the internal source with shape SIN, SQU, RAMP, NRAMp, or TRI. This command applies to the 336xx models only.

Parameter Typical Return
From -360 to 360 (DEGree)
From -2π to 2π (DEGree)
From -arbitrary waveform period to +arbitrary waveform period (SECond)
 +3.000000000000000E+01

Set the modulation phase to 30 degrees:
UNIT:ANGL DEG
MOD:PHAS 30
  • The default units of the <angle> parameter are set by UNIT:ANGLe.
