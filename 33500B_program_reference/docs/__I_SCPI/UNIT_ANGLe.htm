UNIT:ANGLe {DEGree|RADian|SECond|DEFault}
UNIT:ANGLe?

Specifies the angle units that displayed on the screen and used for specifying angles. The selected units are used for setting the starting phase for a burst (BURSt:PHASe) and for setting the phase offset (PHASe). The associated queries are also affected.

Parameter Typical Return
{DEGree|RADian|SECond|DEFault}, default DEGree DEG, RAD, or SEC
Set angle units to radians:
UNIT:ANGL RAD
  • The SECond option is only available for the 336xx models.
  • The setting may be overridden by adding units to numeric parameter in command. For example, PHASE 90 DEG specifies 90 degrees, regardless of this setting.
  • The front-panel display always shows degrees, regardless of the UNIT:ANGLe setting.
