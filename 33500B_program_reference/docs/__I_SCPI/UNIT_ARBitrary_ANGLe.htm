UNIT:ARBitrary:ANGLe {DEGree|RADian|SECond|SAMPle|DEFault}
UNIT:ARBitrary:ANGLe?

Sets the units for specifying arbitrary waveform phase. The front-panel will display the phase parameter in the selected units.

Parameter Typical Return
{DEGree|RADian|SECond|SAMPle|DEFault}, default DEGree DEG, RAD, SEC, or SAMP
Set the arbitrary waveform angle units to samples:
UNIT:ARB:ANGL SAMP
  • The SECond and SAMPle options are only available for the 336xx models.
  • You can also specify units by adding units to the numeric parameter: ARB:PHAS 10 DEG.
